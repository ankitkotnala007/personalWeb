xTTchain : Automation

xTTchain is the project based on the Automation. It is an automatic Toll Tax Collection System . In this the picture is clicked of the vehicle at the toll and Using the Optical Character Recognisation technique we used to fetch the vehicle nubmer from the number plate and sending to the server.

For that we have two online portals , One for the user (which is puclic) and one for the Government . The customer will be allready registered with the portal at the time of the purchase of the vehical. So than when the server get the number of the vehical the amount of money will automatically deducted from the account of the customer.

In this we used several Technologies :
- Jsp and Servelets (Creation of the dashboard for both the parties)
- Android (for the use of the camera as a device)
- Html ,css ,javaScript , jquery (for the UI and UX of the portal )
- Android Studio , Eclipse IDE , Sqlite Database
