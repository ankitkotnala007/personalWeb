August 15, 2018 / by Ankit Kotnala

The Tic Tac Toe is a very famous Game . Today we will be implementing that game using the Java swing . Initially you can use any editor of your choice eg . notepad++ , visual studio code , netbeans IDE , Eclipse IDE etc . But for running the code , jdk (Java Devlopment Kit) and jre (Java Runtime Environment) should be installed in your system .
if you dont have jdk and jre installed go through the link :

                        private void retry() {
                          int x = 0;
                          int dialogButton = JOptionPane.YES_NO_OPTION;
                           x = JOptionPane.showConfirmDialog(null,"Do you like to Retry ?","",dialogButton);
                            dispose();
                          if(x == 0) {
                             main(null);
                          }
                      }
